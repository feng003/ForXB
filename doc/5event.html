  • 触摸类：onTouchCancel / onTouchEnd /onTouchMove / onTouchStart

  • UI元素：onScroll
  • 滚动：onWheel
  • 鼠标：onClick / onContextMenu（右键） / onDoubleClick / onMouseDown / onMouseEnter(Leave Move Out Over Up)
  • 拖拽：onDrop / onDrag (onDragEnd Enter Leave Over)
